  <em>Marketing site for <a href="https://github.com/nitaybz/ginnie-agents">ginnie-agents</a> — the open-source framework for AI teammates that live in your Slack.</em>
</p>

---

A small, static, dark-themed Next.js site. Pairs Fraunces with Geist Mono. Deployed on Vercel.

## Develop

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Stack

- Next.js 16 (App Router, Turbopack)
- React 19
- Tailwind CSS v4
- Fonts via `next/font/google` (Fraunces, Geist Mono)

## Deploy

Auto-deploys to Vercel on push to `main`.
